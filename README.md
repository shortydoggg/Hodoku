# Hodoku version 2.2
![Image2](https://github.com/user-attachments/assets/4d3fb927-a784-4373-ac46-1a0268fea513)

Hodoku is a Sudoku playing/learning Java application that can be compiled into a portable jar file.  

The jar file can be ran by the recommended command from inside the folder containing the jar file:    

    java -Xmx512m -jar hodoku.jar 
where 512 megabytes is the recommended heap size. 

### SYSTEM REQUIREMENTS:

    Windows XP, Vista, 7, 8, or 10 (for exe files) or Linux - Ubuntu/GTK+-LAF
        
    Java Runtime Environment (JRE) 1.6 or above
